Acerca de Solecito

Solecito es el portal de tareas del Jardín de Niños Solecito, pensado para que los papás consulten fácilmente las actividades pendientes de sus hijos por grado, y para que los docentes den de alta, editen y eliminen tareas desde un panel propio.

Dirección: Morelia, Michoacán, México
Horario de atención: Lunes a viernes, 8:00 a 14:00 hrs
Contacto: usa el formulario de contacto para escribirnos.

Tecnologías utilizadas

El proyecto está construido con Django como framework principal, utilizando plantillas HTML con herencia de bloques, un sistema de autenticación para el panel docente, y una base de datos relacional para el manejo de tareas, dudas y respuestas.
